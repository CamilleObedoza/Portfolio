Excel Project

An interactive Excel dashboard for a team that didn't have access to Power BI, built with pivot tables and filters so it still felt easy to use.

Excel Pivot Tables Slicers

Overview

Not every team had a BI tool license, but they still needed a clear, interactive way to review their data instead of manually filtering raw spreadsheets. I structured the raw data into a clean, single flat table first, then built pivot tables and charts on top, with slicers layered in for interactive filtering.

Once the data was filterable by category and time period, seasonal patterns that had been buried in the flat, unfiltered view became obvious for the first time.
